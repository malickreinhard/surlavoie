Le contexte

La loi sur l'égalité pour les personnes handicapées (LHand) imposait aux transports publics suisses d'être entièrement accessibles d'ici fin 2023. Après 20 ans de délai, de nombreuses gares et trains ne sont toujours pas conformes. Les CFF ont obtenu des prolongations jusqu'en 2036, voire au-delà.

Cette initiative

Ce projet est porté par la newsletter « Couper l'herbe sous les roues », dans une démarche de journalisme de solutions.

Objectifs :

  • Quantifier les difficultés rencontrées par les personnes handicapées lors de leurs trajets CFF.
  • Systématiser les réclamations pour obtenir des réponses et des améliorations.

Comment ça fonctionne

  1. Vous remplissez le formulaire avec les détails de l'incident.
  2. Un courrier formel est généré automatiquement (PDF).
  3. Vous l'envoyez vous-même à la direction des CFF par courrier postal ou e-mail.
  4. Les CFF sont tenus de vous répondre.

À savoir

  • Cet outil n'est ni soutenu ni développé par les CFF ou une organisation gouvernementale.
  • Il ne constitue pas un conseil juridique et ne remplace pas une avocate ou un avocat.
  • Version bêta : des bugs peuvent subsister.
  • Une suggestion ou un bug ? Le bouton « Faire un feedback » en bas de page est là pour ça.

          Déclaration d'accessibilité

          Déclaration d'accessibilité de Mondame Productions pour Handicapé sur la voie.

          Mesures en faveur de l'accessibilité

          Mondame Productions prend les mesures suivantes pour assurer l'accessibilité de Handicapé sur la voie :

          • Inclure l'accessibilité dans notre énoncé de mission
          • Intégrer les personnes handicapées dans nos personas de conception
          • Impliquer des personnes directement concernées dans le développement
          • Utiliser des outils d'évaluation automatisée de l'accessibilité

          Statut de conformité

          Les Web Content Accessibility Guidelines (WCAG) définissent des exigences pour améliorer l'accessibilité pour les personnes handicapées. Elles définissent trois niveaux de conformité : A, AA et AAA.

          Handicapé sur la voie est entièrement conforme au WCAG 2.1 niveau AA. Cela signifie que le contenu est pleinement conforme à la norme d'accessibilité, sans aucune exception connue.

          Spécifications techniques

          L'accessibilité de Handicapé sur la voie repose sur les technologies suivantes :

          • HTML sémantique
          • CSS avec respect des préférences utilisateur (prefers-reduced-motion, contrastes)
          • JavaScript accessible (gestion du focus, attributs ARIA)
          • Widget d'accessibilité UserWay
          • Génération PDF avec structure accessible

          Ces technologies sont utilisées en conformité avec les normes d'accessibilité en vigueur.

          Approche de l'évaluation

          Mondame Productions a évalué l'accessibilité de Handicapé sur la voie à l'aide des approches suivantes :

          • Auto-évaluation par une personne directement concernée par le handicap
          • Tests avec lecteurs d'écran
          • Validation automatisée via UserWay
          • Vérification des contrastes WCAG

          Politique de confidentialité

          Politique de confidentialité de Mondame Productions pour Handicapé sur la voie.

          Date d'entrée en vigueur : 6 décembre 2025

          Introduction

          Mondame Productions (« nous », « notre ») exploite l'outil Handicapé sur la voie, accessible via surlavoie.online. Nous nous engageons à protéger votre vie privée et à traiter vos données personnelles conformément à la Loi fédérale sur la protection des données (LPD) et au Règlement général sur la protection des données (RGPD).

          Données collectées

          Données que vous fournissez volontairement :

          • Informations d'identité (nom, prénom, adresse, email, téléphone) — uniquement si vous choisissez de ne pas rester anonyme
          • Informations sur votre voyage (date, gares, horaires, numéro de train)
          • Description de l'incident d'accessibilité

          Données collectées avec votre consentement explicite :

          • Données factuelles anonymisées de l'incident (date, lieu, type de problème) — transmises à des fins de documentation statistique uniquement si vous cochez la case de consentement

          Données techniques :

          • Données de navigation anonymes via le widget UserWay (accessibilité)

          Traitement local des données

          Important : L'ensemble du traitement de vos données personnelles s'effectue localement dans votre navigateur. Vos informations personnelles (nom, adresse, etc.) ne sont jamais transmises à nos serveurs.

          • Le formulaire et le PDF sont générés entièrement côté client (dans votre navigateur)
          • Vos données sont temporairement stockées dans le localStorage de votre navigateur pour permettre la sauvegarde automatique du brouillon
          • Ces données sont supprimées lorsque vous cliquez sur « Recommencer » ou effacez les données de votre navigateur

          Finalités du traitement

          • Génération de votre réclamation : création du PDF contenant votre lettre de réclamation
          • Documentation statistique (optionnelle) : collecte anonymisée des incidents pour documenter les problèmes d'accessibilité sur le réseau CFF

          Partage des données

          Nous ne vendons, ne louons ni ne partageons vos données personnelles avec des tiers, à l'exception de :

          • Données anonymisées : si vous y consentez, les données factuelles anonymisées de l'incident sont stockées via Google Sheets à des fins de documentation
          • UserWay : widget d'accessibilité (aucune donnée personnelle transmise)

          Vos droits

          Conformément à la LPD et au RGPD, vous disposez des droits suivants :

          • Droit d'accès à vos données
          • Droit de rectification
          • Droit à l'effacement (« droit à l'oubli »)
          • Droit à la limitation du traitement
          • Droit à la portabilité des données
          • Droit d'opposition

          Pour exercer ces droits, contactez-nous à : [email protected]

          Sécurité

          Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données, notamment :

          • Connexion HTTPS chiffrée
          • Traitement local sans transmission de données personnelles
          • Aucun stockage de données personnelles sur nos serveurs

          Conservation des données

          • Données locales (navigateur) : supprimées automatiquement après 24 heures
          • Données anonymisées (si consentement) : conservées indéfiniment à des fins statistiques et de plaidoyer
